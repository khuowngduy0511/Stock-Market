## Run Locally

Environment:
 - Operating System: Window, Linux.
 - [.NET 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

Clone the project

```bash
  git clone https://github.com/khuowngduy0511/Stock-Market.git
```

Go to the project directory

```bash
  cd Stock-Market
  cd web-api-example
```

Start the project

```bash
  dotnet watch run
```
